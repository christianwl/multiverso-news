# 🎮 Multiverso News

O **Multiverso News** é um projeto de estudo desenvolvido originalmente no início do programa **Proprofissão (Instituto PROA)**. O objetivo principal foi aplicar conceitos fundamentais de HTML e CSS na construção prática de um portal de notícias de games.

## 📝 Sobre o Projeto
O site é um modelo de página inicial estruturado de forma semântica:
* **Header:** Cabeçalho com navegação.
* **Main:** Seção principal com 3 notícias do mundo dos jogos.
* **Footer:** Rodapé com informações de propriedade intelectual.
* **Dark Mode:** Implementação de um modo escuro para alternância de temas.

## 🚀 Evolução do Projeto

### Início de 2024 (Versão Inicial)
* Criação da estrutura base em HTML e CSS.
* Primeiro contato com layout de notícias e tags semânticas.
* Implementação da lógica inicial de modo escuro.

### Atualização em 2025 (Melhorias)
* **Nova Logo:** Adição de uma identidade visual autoral.
* **Refatoração de Código:** Melhorias no CSS do Dark Mode.
* **Organização:** Criação de pastas para separar arquivos (CSS, Imagens, etc), facilitando a manutenção.
* **Conteúdo:** Atualização e revisão dos textos das notícias.

### Atualização em 2026 (Melhorias)
* **Refatoração:** Alterar a lógica do botão de "onClick" no html para "addEventListener" no JavaScript via id da tag.

## 🛠️ Tecnologias Utilizadas
* **HTML5** (Tags semânticas)
* **CSS3** (Estilização e modo escuro)
